# Raccolte di specie vegetali — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Homepage preview (4 random photos, 2×2) + `/raccolta` with month filters, grouped «Tutte» view, captions from filenames, cream/green styling, lightbox.

**Architecture:** `buildRaccoltaManifest()` from `src/lib/content/raccolta/{month}/` at build time. `RaccoltaCard` for image+caption. `RaccoltaGallery` renders grouped month blocks or single grid. Reuse shared UI components like Stampe/Gallery.

**Tech Stack:** SvelteKit 5, TypeScript, Vitest, CSS custom properties

**Spec:** `docs/superpowers/specs/2026-05-29-raccolta-design.md`
